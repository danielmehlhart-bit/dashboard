🔧 Zoho Desk API — OAuth Setup

3. März 2026 · Support Dashboard Projekt · IN PROGRESS
Ziel: Read-only Zugang zur Zoho Desk API für ein Support-Dashboard. Tickets, Agent Performance, Trends, SLA-Metriken — alles was deine Leader brauchen.

1 API Console öffnen

Öffne https://api-console.zoho.eu

EU-Domain weil ihr eine europäische Zoho-Instanz habt. Check: Steht in deinem Zoho-Login accounts.zoho.eu? Dann .eu.

2 Self Client anlegen

  1. Klick "Add Client"
  2. Wähle "Self Client" (schnellster Weg für interne Tools)
  3. Bestätige

Du bekommst:

  • Client ID — kopieren!
  • Client Secret — kopieren!
📝 Beide sofort sicher speichern! Brauchst du in Schritt 5.

3 Scopes definieren

Diese Scopes brauchst du für das Support-Dashboard:

Desk.tickets.READ,Desk.contacts.READ,Desk.tasks.READ,Desk.agents.READ,Desk.departments.READ,Desk.reports.READ,Desk.settings.READ
ScopeGibt dir
Desk.tickets.READAlle Tickets, Status, SLA, Tags
Desk.agents.READAgent-Profile, Zuordnungen
Desk.reports.READBuilt-in Reports, Metriken
Desk.departments.READAbteilungen / Teams
Desk.contacts.READKundenkontakte
Desk.settings.READSLA-Policies, Business Hours
Desk.tasks.READAufgaben / Follow-ups

4 Grant Token generieren

Im Self Client Panel:

  1. Scope einfügen (den String von oben, alles in eine Zeile)
  2. Time Duration: 10 Minuten
  3. Scope Description: DRMS Support Dashboard — read-only
  4. Klick "Create"
⚠️ Der Grant Token ist nur 10 Minuten gültig! Sofort weiter zu Schritt 5.

5 Grant Token → Refresh Token tauschen

Sofort nach dem Grant Token — im Terminal oder Postman:

curl -X POST "https://accounts.zoho.eu/oauth/v2/token" \
  -d "grant_type=authorization_code" \
  -d "client_id=DEINE_CLIENT_ID" \
  -d "client_secret=DEIN_CLIENT_SECRET" \
  -d "code=DEIN_GRANT_TOKEN"

Response:

{
  "access_token": "1000.xxxx...",
  "refresh_token": "1000.yyyy...",
  "expires_in": 3600,
  "token_type": "Bearer"
}
📝 Den refresh_token sicher speichern! Der läuft nicht ab (solange du den Client nicht löschst).

6 Testen

Quick-Test ob alles funktioniert:

curl -X GET "https://desk.zoho.eu/api/v1/tickets?limit=5" \
  -H "Authorization: Zoho-oauthtoken DEIN_ACCESS_TOKEN" \
  -H "orgId: DEINE_ORG_ID"
orgId findest du unter: Zoho Desk → Setup → API → Org ID.
Oder via API: GET https://desk.zoho.eu/api/v1/organizations
