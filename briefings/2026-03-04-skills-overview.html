🟢 Eigene Skills (workspace/skills/)
📊 dashboard-update Neu · heute
Dashboard-Briefings, HTML-Updates, Morning Flow
  • Wann: Jedes Mal wenn etwas aufs Dashboard soll — Briefings, TODOs, Leads, Meetings, Trail
  • Was es macht: Definiert den kompletten Flow: HTML erstellen → Dashboard-Link updaten → git push → kurzer Telegram-Ping
  • Enthält: Section-Referenz (welche CSS-Klassen wo), Design-Regeln, Briefing-Typen, Sub-Agent-Regeln
  • Nutzen: Sub-Agents wissen sofort wo Dateien liegen und wie das Dashboard aufgebaut ist — spart ~500 Tokens pro Prompt
🔍 adversarial-review Neu · heute
3-Agent Code Review: Finder → Challenger → Referee
  • Wann: Nach nicht-trivialen Code-Änderungen, neuen Features, Security-relevanten Changes
  • Was es macht: 3 Sub-Agents mit spieltheoretischen Anreizen (Finder bekommt Punkte fürs Finden, Challenger verliert doppelt für falsches Widerlegen)
  • Timeout: 2 Min pro Agent, 8 Min total — nie endlos loopen
  • Output: HTML-Report mit Findings-Tabelle und Accuracy-Score
🎨 gamma-deck Neu · heute
Präsentationen via Gamma API erstellen
  • Wann: Pitch Decks, Follow-Up Decks, Client-Präsentationen
  • Was es macht: Gamma API aufrufen (Auth, Parameter, Status-Polling), bei Fehler Fallback zu HTML-Deck
  • Enthält: Komplette API-Doku (Endpoint, Auth-Header, textMode, imageOptions, sharingOptions)
  • Wichtig: Auth ist X-API-KEY Header, NICHT Bearer Token!
