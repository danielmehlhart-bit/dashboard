Design System Änderungen — Begründungen
BereichAltNeu + Begründung
Layout Linke Sidebar (220px) + 3 Spalten Top Nav (52px) + volle Breite. 168px mehr Content-Platz.
Hintergrund #020617 (Slate-950, kalt-blau) #0d0f12 (wärmer, fast neutral). Weniger ermüdend bei langem Arbeiten.
Cards 8px Radius, harter Border 12px Radius, #1e2128 Border + Box-Shadow. Moderne, freundlichere Anmutung.
Nav Active "Active" Badge Text + Farbe Pill-Hintergrund + Accent Icon. Cleaner, weniger Clutter.
Briefings-Liste 200px fest, Titel truncated 280px default, 180–450px drag. Titel voll sichtbar, Datum als sekundäre Info.
Kategorien Sidebar-Navlinks für Kategorien Horizontal Filter-Chips über der Liste. Schneller, platzsparender.
Preview Dark background für HTML-Content Weißer Hintergrund — HTML-Briefings sind für hell designed.
Home Screen Fehlte / rudimentär Hero + Stat-Cards + Quick Access + Briefing-Cards. Zentraler Ausgangspunkt.
📱 Mobile — Zwei Zustände

Auf Mobile entfallen Labels aus der Nav und das Split Pane. Tippen auf ein Briefing öffnet es direkt fullscreen.
